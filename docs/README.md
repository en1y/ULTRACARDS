# ULTRACARDS

---

## About Ultracards
- It is a Java project that aims to provide a online platform for playing various card games
- You can try it online on the web at [ultracards.en1y.qzz.io](https://ultracards.en1y.qzz.io/) 
- The project is open-source and welcomes contributions from the community
- The main goal of ULTRACARDS is to be a stable and user-friendly platform where players can enjoy themselves.
- The project is designed to be modular, allowing for easy addition of new games and features.

## Current status
- Currently, in the early stages of development with a focus on building a strong OOP foundation for the project
- The project is based on a client-server logic with a strong backend api
- The project already has a web ui where players can play briskula online

## Project plans
- It is to support various card games like _Tresette_, _Poker_, _Durak_, and more.
- It is to be built using JavaFX for the user interface, making it cross-platform compatible.
- The project will also include features such as matchmaking, and searchable player profiles.

## Modules
- `server`: Spring-based backend, web UI, API, authentication, and multiplayer coordination
- `game-gateway`: shared DTOs and integration layer between game logic and clients
- `game-logic`: reusable card and game implementations
- `ui`: client-side UI modules
